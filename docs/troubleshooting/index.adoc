=== Quick Diagnosis

==== Error: "Target already exists"

Remove the existing target or use a different path:

[source,ruby]
----
FileUtils.rm_rf(target)
Excavate::Archive.new(archive).extract(target)
----

==== Error: "Unknown archive format"

Check the file extension:

[source,ruby]
----
puts File.extname(archive_path)
# Should be .zip, .tar, .msi, etc.
----

==== Error: "File not found"

Verify the file exists:

[source,ruby]
----
puts File.exist?(archive_path)
puts File.size(archive_path)
----
