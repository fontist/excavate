=== Quick Start for Contributors

[source,bash]
----
# Clone repository
git clone https://github.com/omnizip/excavate.git
cd excavate

# Install dependencies
bundle install

# Run tests
bundle exec rspec

# Run linter
bundle exec rubocop
----

=== Architecture Overview

Excavate follows object-oriented design principles:

* Facade pattern (`Excavate::Archive`)
* Strategy pattern (format-specific extractors)
* Registry pattern (extension mapping)

See <<architecture/index,Architecture>> for detailed documentation.
