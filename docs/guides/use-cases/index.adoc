=== Quick Reference

.Extract fonts from MSI
[source,ruby]
----
Excavate::Archive.new('fonts.msi').extract(
  recursive_packages: true,
  filter: '**/*.{ttf,otf}'
)
----

.Analyze installer contents
[source,ruby]
----
Excavate::Archive.new('installer.msi').files(recursive_packages: true) do |path|
  puts path
end
----

.Batch process archives
[source,ruby]
----
Dir.glob('*.zip').each { |f| Excavate::Archive.new(f).extract }
----
