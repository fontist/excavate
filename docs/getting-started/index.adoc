=== Key Concepts

Before diving in, understand these core concepts:

.Archive
A single file containing one or more files with metadata.

.Recursive Extraction
Automatically extracting archives within archives (e.g., MSI → CAB → files).

.Selective Extraction
Extracting only specific files from an archive.

.Filter Pattern
Glob pattern for matching multiple files (e.g., `**/*.ttf`).

=== Pure Ruby

Excavate is a 100% pure Ruby gem:

* No native extensions or compiled dependencies
* Works on MRI, JRuby, and TruffleRuby
* Cross-platform: Linux, macOS, Windows, BSD
