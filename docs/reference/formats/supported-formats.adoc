=== Compression Support

==== CAB Compression

* None (uncompressed)
* LZSS (simple dictionary)
* MSZIP (deflate-compatible)
* LZX (high ratio)
* Quantum (memory-efficient)

==== 7-Zip Compression

* LZMA
* LZMA2
* PPMD
* BCJ (x86 filter)
* BCJ2 (enhanced x86)

==== ZIP Compression

* Store (none)
* Deflate

=== Platform Support

.All Platforms:: ZIP, TAR, TAR.GZ, TAR.XZ, 7-Zip
.Windows:: CAB, MSI
.Linux:: RPM, TAR, TAR.GZ, TAR.XZ
.macOS:: PKG, XAR

=== Dependencies

All formats are supported through pure Ruby libraries:

* Omnizip (~> 0.3.8)
* Cabriolet (~> 0.2.2)

No native dependencies required.
