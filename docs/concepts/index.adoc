=== Key Concepts

.Archive:: A single file containing one or more files with metadata
.Extractor:: A class that handles extraction for a specific format
.Recursive Extraction:: Automatically extracting archives within archives
.Format Detection:: Identifying archive type by extension or content
.Excavate Path:: A virtual path that traverses through nested archives
