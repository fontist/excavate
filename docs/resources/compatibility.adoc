=== Known Limitations

==== Windows

* Very long paths (>260 chars) may require special handling
* Some anti-virus software may slow extraction

==== macOS

* PKG files with custom install scripts cannot execute scripts

==== Linux

* No limitations known

=== Testing Coverage

Excavate is tested on:

* Ruby 2.7, 3.0, 3.1, 3.2, 3.3 (MRI)
* JRuby 9.4
* TruffleRuby 22+
* Ubuntu, macOS, Windows

=== Reporting Issues

If you encounter compatibility issues:

. Report Ruby version: `ruby -v`
. Report OS version
. Provide sample archive
. Include error message and backtrace
